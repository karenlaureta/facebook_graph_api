User Data Deletion Instructions

This page explains how users can request deletion of their data associated with this Facebook application.

How to Request Data Deletion

  1. Send an email to karenevangelistalaureta@gmail.com
  2. Use the subject line: Facebook Data Deletion Request
  3. Include your Facebook User ID and App Name

After verification, all associated data will be permanently deleted within 7 business days.

If you have questions, please contact the app administrator.
