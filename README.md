# Facebook Graph API Demo

A simple web app demonstrating **Facebook Graph API** integration to fetch and display basic user profile information.

---

## Features

- Login with Facebook via OAuth
- Display user **name, profile picture, and ID**
- Dark mode toggle
- Loading and error handling
- Privacy & data deletion support

## Usage

1. Open `index.html` in a modern browser.
2. Click **Continue with Facebook**.
3. Grant permissions to fetch your public profile.
4. Your profile info appears in a card.

---

## Setup

- Replace `APP_ID` in `script.js` with your **Facebook App ID**.
- Update `REDIRECT_URI` to match your local or deployed URL.
- Ensure your app has **public_profile** permission.

---

## Privacy

- Only basic public profile info is collected.
- No data is shared or sold.
- Users can request deletion via `delete.html`.
